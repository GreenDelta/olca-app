What's new

For the 1.4 version, we completely redesigned and “renovated” openLCA. Results are

  • drastically improved performance of about 400%,
  • numerous usability improvements, and
  • many new features.

Find out what is new in version 1.4.1 here.

Getting started

With the new version of openLCA, it is really easy to get started you can just import a database from an existing zolca file and open it in the navigation window with a simple double-click.

Some basic openLCA functions are also explained in this video.

Download

A broad range of LCA databases are available in openLCA Nexus, where you can search for both single data sets and complete databases and download them for import into openLCA. The databases are all fit for openLCA and several databases can be combined.

How you can import databases and impact assessment methods into openLCA is also explained in a video.

LCIA method packs are available from the openLCA download page.

Manuals

Find out how to work with openLCA in our User Manuals. These illustrate the most basic features and functions as well as offer comprehensive information on more advanced topics.

Case Studies

Learn by doing with the help of openLCA Case Studies available online for free for versions 1.3 and 1.4. These were created for instructional purposes and can guide you through working with openLCA using practical examples.
